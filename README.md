**Shifrlovchi** is a modern web application for **securely encrypting and decrypting messages** with password protection. Designed with an interactive rotating gears animation, it provides both functionality and a visually engaging experience, making encryption feel sophisticated and accessible.

## Features

- **Password-Protected Encryption & Decryption**  
  Safely encode messages and reveal them only with the correct password.

- **Interactive Gear Animation**  
  Watch the gears turn while encrypting/decrypting, creating an elite, professional user experience.

- **Multi-Language Ready**  
  Easily adaptable to multiple languages and text formats.

- **Lightweight and Fast**  
  Built with performance and simplicity in mind.
